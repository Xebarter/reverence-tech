## 🎨 BLOG POST FORMATTING STANDARDS

### Every Post Must Include:

#### SEO Elements:
- [ ] Target keyword in H1 (title)
- [ ] Keyword in first 100 words
- [ ] Keyword in URL slug
- [ ] Keyword in meta description
- [ ] 2-3 H2 headers with keyword variations
- [ ] Alt text for all images (keyword-rich)
- [ ] Internal links (3-5 per post)
- [ ] External links (2-3 authority sites)
- [ ] FAQ section at end (schema markup)

#### Content Elements:
- [ ] Compelling introduction (problem/hook)
- [ ] Table of contents for long posts
- [ ] Bullet points & numbered lists
- [ ] Images every 300-400 words
- [ ] Data & statistics (cited)
- [ ] Examples & screenshots
- [ ] Conclusion with key takeaways
- [ ] Strong CTA (quote request, consultation, etc.)

#### Images:
- [ ] Featured image (1200x630px for social sharing)
- [ ] 5-10 supporting images
- [ ] WebP format
- [ ] Compressed (<100KB each)
- [ ] Descriptive file names (keyword-rich)
- [ ] Alt text for all images

#### Schema Markup:
- [ ] Article schema
- [ ] Author schema (E-E-A-T)
- [ ] FAQ schema (if applicable)
- [ ] BreadcrumbList schema
- [ ] ImageObject schema

## 🎓 WRITING GUIDELINES

### Tone & Style:
- **Professional but approachable**
- Use Ugandan English spelling (colour, organisation)
- Reference local context (Kampala, mobile money, etc.)
- Address pain points directly
- Provide actionable advice
- Avoid jargon (or explain when used)

### Formatting:
- Short paragraphs (3-4 sentences max)
- Use subheadings every 300 words
- Bullet points for lists
- Bold for emphasis (sparingly)
- Block quotes for important takeaways
- White space for readability

### Examples & Data:
- Use real numbers from Uganda market
- Reference local businesses (with permission)
- Include East Africa statistics
- Cite government sources (NITA-U, UBOS)
- Link to authority sites

### Calls-to-Action:
- One primary CTA per post
- Place at natural break points
- Use action verbs ("Get", "Start", "Request")
- Create urgency ("Limited slots", "Free this month")
- Make it relevant to content
